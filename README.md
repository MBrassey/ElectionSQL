## ElectionSQL

This application uses sqlite3 queries and express.js routes to create a working election node back end.

![licensebadge](https://img.shields.io/badge/license-CC0_1.0_Universal-blue)

#### Requirements

- node.js
- npm

#### Installation

    npm i

#### Usage

    npm run migrate
    npm run seed
